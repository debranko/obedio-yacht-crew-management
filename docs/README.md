## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript + Prisma
- **Database:** PostgreSQL 14+
- **Real-time:** Socket.io (WebSocket)
- **Hardware:** ESP32-S3 Smart Button
